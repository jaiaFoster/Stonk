# Mobile Friendly UI v1

Lightweight UI patch focused on mobile usability and low-risk report improvements.

## Changes

- Adds a mobile-friendly root menu at `/`.
- Adds `meta viewport` tags to report and loading pages.
- Adds top quick navigation for major report sections.
- Moves Pipeline Status down into the Debug / Copyable Output area.
- Makes tables horizontally scrollable on mobile.
- Improves mobile spacing, button sizing, and debug/log readability.

## Why

The report has become useful but dense. This keeps the full information available while making the page easier to navigate on a phone.

## No strategy changes

This patch does not change the scoring, earnings discovery, portfolio gap, trade memory, or daily opportunity logic.
